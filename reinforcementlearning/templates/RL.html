Reinforcement Learning

{% mathjax_scripts %} Reinforcement learning is a type of machine learning in which a series of agents that interact with an environment in order to maximise their reward. The agents observe a set of states and carry out a series of actions. $$1+1=2$$

The Q-Learning Algorithm

In the Q-learning algorithm the agent attempts to determine the best action to take for each state the agent may be in.
